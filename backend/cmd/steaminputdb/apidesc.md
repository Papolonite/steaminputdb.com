[_SteamInputDB.com_](https://steaminputdb.com) is a community-driven database of Steam Input configurations for various games,
**backed by the official Steam Web API**.

[_SteamInputDB_](https://steaminputdb.com) allows users to share, discover,
and download custom controller configurations to enhance their gaming experience on Steam.

This API allows developers and users to access and interact with the database programmatically.
